# 🏎️ Autóverseny

C# nyelven írt autóverseny szimulációs játék.

## Leírás

Egy egyszerű autóverseny játék C# nyelven. Az autók versenyeznek egymás ellen, a játékos irányíthatja a saját autóját.

## Funkciók

- Autó irányítás
- Versenypálya szimuláció
- Pontozó rendszer

## Technológiák

- **Nyelv:** C#
- **.NET** keretrendszer

## Használat

1. Klónozd a repót:
   ```bash
   git clone https://github.com/Kristof323/autoverseny.git
   ```
2. Nyisd meg Visual Studio-ban
3. Fordítsd és futtasd
